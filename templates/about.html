About Us

Green Plate was developed by using the IP Geolocation API and the Google Maps API in order to get the user's IP address and derive a location from that information. This location, in latitude and longitude, is given to the Google Maps API which then provides information regarding the restaurants in the nearby area. These restaurants are then looked up in a curated database that stores information regarding the restaurants carbon footprint. This information is then displayed to the user!
